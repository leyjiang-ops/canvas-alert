📅 订阅到苹果日历

订阅后 DDL 自动出现在 iPhone / iPad / Mac 的日历里，已提交的会自动消失。

📲 iPhone / Mac 一键添加 🌐 查看 .ics 原文

用 iPhone 相机扫码 → 自动打开订阅

iPhone / iPad

用相机扫码或点击"一键添加" → 弹出"添加订阅日历" → 确认。

macOS

日历 App → 文件 → 新建日历订阅 → 粘贴 URL → 自动更新选 15 分钟。

Google Calendar

设置 → 添加日历 → 通过网址添加 → 粘贴 https:// URL（不是 webcal）。

⚠️ iOS 自动拉取最快约 15 分钟一次（系统控制），手动下拉刷新可立即同步。calendar_id 使用随机串可提升隐私。

🚀 部署步骤

只需三步，配置完成后全自动运行。

1

Fork 仓库

访问 canvas-alert，点右上角 Fork。

2

添加 Secret

在 fork 的仓库里：Settings → Secrets → Actions → New secret，Name 填 USERS_CONFIG，Value 粘贴上方 JSON。

3

启用 Actions

进入 Actions 标签页，启用 workflows，手动触发「全量同步」验证是否收到通知。

你的 token 仅在本地处理，不经过任何服务器。

如何获取 Canvas Token

  1. 登录你学校的 Canvas
  2. 点右上角头像 → Account / Settings
  3. 滚动到 Approved Integrations
  4. 点 New Access Token
  5. 填写用途描述，点生成，复制 token

⚠️ Token 只显示一次，请立刻复制保存。
